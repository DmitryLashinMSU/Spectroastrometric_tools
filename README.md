# Spectroastrometric Tools

Here you can find graphical user interfaces for processing and planning spectroastrometric observations: 
* **SPARTA** (Spectral Processing for Analysis, Reduction and Two-source Astrometry) 
* **TALOS** (Test for Astrometric Limits of Observed Spectra).

### Setup

For Linux executables, you must grant execution permissions before running them. Navigate to the directory containing the file and execute the following command:

```bash
chmod +x filename
```
In Windows, select the appropriate resolution in the dialog box that appears when you run the file.
